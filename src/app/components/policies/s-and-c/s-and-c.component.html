Security and Privacy Policy

Within this Privacy Policy, all information regarded data collected from the user, the reasoning behind the collection of data, ways this data will be used, who will have access to this data as well as what will occur in the event of a takeover till be disscussed on this page

Collected Information

All information collected from the user such as the users, username, email, passwrod, and timetable will be stored. If the user decided to login using Google, the login credentials will also be collected and stored


Reasoning For Collecting Data

All information stored is to generate user accounts and authenticate them in the future. This will allow the user to access saved work that will be stored in association with their account


Use of Data

All data collected from the user is to inhance user experience. Login credentials are stored so users can access information within their account


How this Information is Stored

All user credentials is stored within a Google Firebase. All information regarding the users time table is stored locally within the system


Who has access to this information

The only individuals that have access to this information are the Site Manager, as well as Google and Firebase


In the case of takeover

In the case of takeover, a new privacy policy might be created and in that case it will be sent out the all users
